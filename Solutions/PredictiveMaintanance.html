Predictive Maintanance Solution

The Intelligence beyond the Telematics

Nextgen Predictive Maintenance Solution

Our PLACER SOLUTION , driven by cutting edge analytics AI and Machine learning, provides real time predictive maintenance solution
  • Parameters captured from a Machines and Buildings over time and predict the need for maintenance before something go wrong.
  • The data captured from Assests in real-time is processed and analysed by analytics module built for identifying certain patterns in the Machine parameters received and notify the users about the need for service.

Machines

  • Live Tracking
  • Dashboard for Tracking, Logs, Reports, Analysis, Alerts, Configurations and Admin
  • Event Based Notifications – eg. Speed, Halt, Movement, Reached, Left etc..
  • Performance Parameters – Units Produced, Length cut, etc…
  • Engine Performance – RPM, Coolant Temperature, Load, Fuel Temperature, Fault Codes
  • Electrical – Voltage, Amps, Load, Units, etc..
  • Gear – RPM, Fault codes, Performance, Output Shaft speed etc..




Buildings

  • Live Tracking
  • Dashboard for Tracking, Logs, Reports, Analysis, Alerts, Configurations and Admin
  • Integration facilities to any Building Maintenance Software
  • Facility to track the vehicle by the people on the emergency contact list
  • Indoor Air Quality – Temperature, Humidity, Co2 etc..
  • No. of Occupants
  • Event Based Notifications – Temperature Range, Co2 level etc..
